## 프로젝트명 (Project Name)

팀 기반 To-Do List 웹 애플리케이션

---

## 기술 스택 (Tech Stack)

- Frontend: React, TypeScript
- Backend: Express, TypeScript
- Database: MariaDB
- Build Tool: Vite
- HTTP Client: Axios
- Version Control: Git

---

## 주요 기능 (Features)

- 사용자 로그인 및 회원가입
- 개인 및 팀 기반 투두리스트 작성
- 할 일 추가, 수정, 삭제
- 팀 생성 및 초대
- 팀원 관리 (초대/삭제)
